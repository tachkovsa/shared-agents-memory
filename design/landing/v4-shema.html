Как читать схему

Четыре этапа потока памяти

01 · agents

Агенты

Claude Code, Codex, OpenCode и внутренние copilot-сервисы пишут и читают память.

02 · api

ArtelMemory API

Единый вход: cookie-сессия и CSRF на мутациях. MCP и REST.

03 · pipeline

Pipeline

Дедупликация, ранжирование (decay), staleness-аудит и lifecycle памяти.

04 · storage

Storage

Qdrant для векторов, SQLite для метаданных, локальные embeddings.

Данные под контролем

Local-only режим: документы и embeddings не покидают ваш контур.

Семантический поиск

Qdrant с payload-фильтрацией по namespace и RU-контекстом.

Общая память команды

Агенты, люди и инструменты работают в едином контексте.
