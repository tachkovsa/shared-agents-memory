Quickstart
artelmemory — shared memory for agents

Память для агентов
в одну команду

Self-hosted слой памяти с семантическим поиском. Поднимается через Docker Compose, подключается к Claude Code, Codex и MCP-клиентам, индексирует локально.

$ docker compose up artelmemory
